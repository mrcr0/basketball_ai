基于AI的科学智能篮球训练Web平台 毕设整体框架与大纲
适配  网页浏览器访问  ，B/S架构、前后端分离，软件工程专业标准毕设结构，可直接用于开题报告、论文撰写、项目设计文档。 
     一、项目概述（适配表格100字简介） 本项目为  基于AI的科学智能篮球训练Web平台  ，采用Vue3 + Spring Boot + MySQL前后端分离架构，集成DeepSeek/豆包大模型，实现球员个人档案管理、AI定制化篮球训练计划、动作识别纠错、战术知识学习、训练数据可视化、篮球社区交流等功能。面向篮球爱好者、学生球员，提供科学、系统化的线上训练指导，解决训练不系统、动作不标准、缺乏针对性方案、战术学习困难等问题。
      二、技术选型（网页端适配，贴合软件工程常用技术栈） 
1.  开发工具  ：VS Code、Trae AI、IDEA 
2.  前端（网页端）  ：Vue3 + Vue Router + Element Plus + ECharts（数据可视化） 
3.   后端  ：Spring Boot、Spring MVC、MyBatisPlus
 4.   数据库  ：MySQL 8.0 5.   AI大模型  ：DeepSeek / 豆包大模型（训练计划生成、动作点评、战术讲解、训练分析） 6.   辅助技术  ：JWT登录鉴权、网页端摄像头动作识别、视频播放、训练报告导出、文件上传      
三、系统总体架构（B/S网页架构） 整体采用  浏览器-服务器(B/S)架构  ，四层结构：
 1.   表现层  ：Web网页前端（用户前台 + 管理员后台，浏览器直接访问）
2.   业务逻辑层  ：Spring Boot后端，处理用户请求、AI接口调用、业务逻辑校验 
3.   数据服务层  ：数据库交互、视频/图片存储、大模型接口调用
 4.   数据持久层  ：MySQL数据库，存储用户信息、训练计划、训练记录、动作库、战术库、社区动态等 
四、功能模块完整大纲（分  用户前台网页  、  管理员后台网页  、  AI核心模块  ）
        （一）用户前台网页模块（普通用户浏览器访问） 
1.  用户账号模块   - 注册、登录、找回密码、完善个人篮球档案（身高、体重、位置：后卫/前锋/中锋、球龄、薄弱项、训练目标、基础水平） - 个人中心、资料修改、训练档案查询、账号安全设置 
2.  AI智能篮球训练计划模块（核心功能）   - 用户录入个人身体条件、场上位置、薄弱技术、训练目标，AI一键生成  周/月科学训练计划   - 计划包含：运球、投篮、传球、防守、体能、战术配合专项训练，可自定义修改、保存、导出PDF - 按天展示训练项目、组数、时长、训练要点、休息时间、训练禁忌 
3.  篮球动作教学与AI纠错模块   - 标准动作库：按技术分类（运球、投篮、突破、防守、传球、脚步），图文+高清教学视频 - 网页端调用摄像头实时识别动作，AI识别投篮手型、运球姿势、脚步、防守站位，给出纠错建议 - 动作难度分级：新手入门、进阶提升、专业强化 
4.  训练打卡与数据可视化   - 按计划打卡训练，记录投篮命中率、运球时长、训练强度、体能消耗、完成度 - ECharts可视化：投篮命中率曲线、训练时长统计、技术短板分析、月度训练报告 - 历史训练记录查询、训练数据对比分析 
5.  AI战术学习与知识问答模块   - AI讲解篮球基础战术、跑位技巧、攻防站位、规则科普 - 在线AI问答，解答战术、技术、伤病防护、体能训练相关问题
6.  篮球社区与资讯模块   - 篮球赛事资讯、球星训练干货、战术科普文章 - 用户发布训练动态、打卡分享、技术交流、评论点赞 
7.   辅助工具模块   - 篮球体能测试、BMI计算、训练日程提醒、训练日历
        （二）管理员后台网页模块（独立后台系统）
 1.   用户管理  ：查看、管理、导出平台用户数据，管理球员训练档案 
2.   动作库管理  ：增删改查篮球技术动作，上传教学视频、编辑训练要点 
3.   训练模板管理  ：管理通用训练计划模板，为AI生成计划提供基础数据 
4.   战术&资讯管理  ：发布、编辑、上下架篮球战术知识、赛事资讯、科普文章 
5.   社区内容管理  ：审核用户动态、评论，处理违规内容 6.   平台数据看板  ：用户量、训练热度、热门训练项目、薄弱技术项统计可视化
        （三）AI大模型核心模块（毕设创新加分点） 1. 基于球员位置、水平、薄弱项，智能生成个性化训练计划 2. 篮球动作识别与姿势纠错、技术点评 3. 篮球战术讲解、训练数据分析、短板提升建议 4. 篮球知识在线问答、训练方案优化 
     五、数据库核心设计（网页端数据表） 
1.   用户表(user)  ：用户ID、账号密码、昵称、身高、体重、场上位置、球龄、训练目标、注册时间 
2.   训练计划表(basketball_plan)  ：计划ID、用户ID、训练周期、AI生成训练内容、专项侧重、创建时间 
3.   训练记录表(train_record)  ：记录ID、用户ID、训练项目、命中率、时长、打卡时间、完成情况 
4.   篮球动作表(basketball_action)  ：动作ID、名称、技术分类、难度、教学视频、标准训练要点 
5.   战术资讯表(tactic_info)  ：资讯ID、标题、战术/科普内容、发布时间、是否上架 
6.   社区动态表(dynamic)  ：动态ID、用户ID、训练内容、发布时间、点赞、评论数
