# 篮球训练平台 - 用户操作手册

## 1. 系统概述

欢迎使用基于AI的科学智能篮球训练Web平台。本平台旨在通过AI技术帮助篮球爱好者提升训练效率，包括训练计划制定、动作分析、实时比赛模拟等功能。

## 2. 用户注册与登录

### 2.1 用户注册

1. 访问首页，点击"注册"按钮
2. 填写注册信息：
   - 用户名：字母、数字组合，3-20位
   - 密码：6-20位字符
   - 昵称：显示名称
   - 位置：擅长的场上位置（后卫/前锋/中锋）
3. 点击"注册"完成注册

### 2.2 用户登录

1. 访问首页，点击"登录"按钮
2. 输入用户名和密码
3. 点击"登录"进入系统

### 2.3 管理员登录

管理员账号拥有全部功能权限：
- 用户名：admin
- 密码：123456

## 3. 训练计划

### 3.1 AI生成训练计划

1. 进入"训练"页面
2. 填写训练需求：
   - 训练目标（提升投篮/运球/传球等）
   - 训练时长（30分钟/60分钟/90分钟）
   - 难度等级（新手/进阶/专业）
3. 点击"生成计划"，AI将生成个性化训练方案
4. 查看训练计划详情，点击"开始训练"

### 3.2 执行训练

1. 在训练计划页面，点击"开始训练"
2. 按照步骤完成每个训练动作
3. 记录训练数据（次数、时间等）
4. 完成训练后提交记录

## 4. 动作库

### 4.1 浏览动作

1. 进入"动作库"页面
2. 按分类筛选（投篮/运球/传球/防守）
3. 按难度筛选（新手入门/进阶提升/专业训练）
4. 点击动作查看详情

### 4.2 动作详情

每个动作包含：
- 动作名称和分类
- 难度等级
- 动作描述
- 技术要点
- 训练建议

## 5. 实时比赛模拟

### 5.1 创建比赛

1. 进入"比赛"页面
2. 点击"创建比赛"
3. 设置比赛参数：
   - 球队名称
   - 比赛时长
   - 难度等级
4. 点击"开始比赛"

### 5.2 观看比赛

1. 比赛开始后，观看实时比赛画面
2. 查看比赛数据统计（得分、篮板、助攻等）
3. 切换视角和显示模式

## 6. 战术知识库

### 6.1 浏览战术

1. 进入"战术"页面
2. 按类型筛选（战术/科普/资讯）
3. 点击文章查看详情

### 6.2 学习战术

- 阅读战术解析文章
- 观看战术示意图
- 查看相关训练建议

## 7. 社区互动

### 7.1 发布动态

1. 进入"社区"页面
2. 点击"发布动态"
3. 输入内容（文字+图片可选）
4. 点击"发布"提交

### 7.2 互动功能

- 点赞：点击动态下方的点赞按钮
- 评论：在评论区输入评论内容
- 分享：分享动态到其他平台

## 8. 个人中心

### 8.1 个人资料

1. 进入"我的"页面
2. 点击"编辑资料"
3. 修改个人信息（昵称、头像、位置等）
4. 点击"保存"

### 8.2 训练记录

1. 进入"我的"页面
2. 点击"训练记录"
3. 查看历史训练数据
4. 查看训练统计图表

### 8.3 修改密码

1. 进入"我的"页面
2. 点击"修改密码"
3. 输入原密码和新密码
4. 点击"确认修改"

## 9. 管理员功能

### 9.1 用户管理

1. 登录管理员账号
2. 进入"用户管理"页面
3. 可进行以下操作：
   - 查看用户列表
   - 修改用户状态（启用/禁用）
   - 设置管理员权限

### 9.2 内容审核

1. 进入"社区内容管理"页面
2. 查看待审核动态
3. 点击"审核通过"或"删除"

### 9.3 数据统计

1. 进入"数据看板"页面
2. 查看系统统计数据：
   - 用户总数
   - 活跃用户数
   - 动态总数
   - 待审核数量

### 9.4 系统日志

1. 进入"登录日志"或"操作日志"页面
2. 查看系统操作记录
3. 支持按条件筛选

### 9.5 数据备份

1. 进入"数据备份"页面
2. 点击"创建备份"
3. 可进行恢复或删除操作

## 10. 常见问题

### 10.1 忘记密码

暂未开放密码找回功能，请联系管理员重置。

### 10.2 无法登录

- 检查用户名和密码是否正确
- 检查网络连接
- 联系管理员确认账号状态

### 10.3 动态无法发布

- 检查内容是否符合规范
- 动态需要审核通过后才能显示

### 10.4 训练计划无法生成

- 检查网络连接
- 检查输入参数是否完整

如有问题，请联系系统管理员获取帮助。
