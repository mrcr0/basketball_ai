## ⚠️ 503错误解决方案

如果您看到"503 - Tunnel Unavailable"错误：

### 解决方案1：刷新页面
```
按 F5 刷新浏览器
等待 5-10 秒后重试
```

### 解决方案2：清除浏览器缓存
```
Chrome: Ctrl + Shift + Delete -> 清除缓存
Firefox: Ctrl + Shift + Delete -> 清除缓存
```

### 解决方案3：使用无痕模式
```
Chrome: Ctrl + Shift + N
Firefox: Ctrl + Shift + P
然后访问: https://ai-basketball-2026.loca.lt
```

### 解决方案4：等待重新启动
如果以上方法都不行，可能需要等待1-2分钟后再试

## 🐛 常见问题

### Q: 为什么显示503错误？
A: 可能是之前的tunnel连接已断开，需要重新启动localtunnel服务

### Q: 为什么访问需要验证？
A: localtunnel的免费服务需要通过浏览器验证来防止滥用

### Q: 可以使用其他内网穿透工具吗？
A: 可以，您也可以尝试：
- ngrok (需要注册)
- frp (需要服务器)
- bore (Rust编写)

### Q: 内网穿透不稳定怎么办？
A: localtunnel免费版有连接数限制，建议：
- 避免频繁刷新页面
- 使用完记得关闭浏览器标签页
- 必要时重新启动服务
