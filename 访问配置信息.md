## 🔄 重新启动穿透服务

### 如果穿透服务断开，按以下步骤重启：

#### 1. 检查服务状态
```powershell
# 检查前端服务
netstat -ano | findstr :5173

# 检查后端服务
netstat -ano | findstr :8080
```

#### 2. 重启前端穿透
```powershell
lt --port 5173
```

#### 3. 重启后端穿透
```powershell
lt --port 8080 --subdomain basketball-backend-8080
```

## 🐛 故障排查

### 503错误
**原因**：穿透服务断开

**解决方案**：
1. 刷新页面 (F5)
2. 等待 5-10 秒
3. 清除浏览器缓存
4. 使用无痕模式

### 无法连接
**原因**：服务未启动

**解决方案**：
1. 检查后端是否运行：http://localhost:8080
2. 检查前端是否运行：http://localhost:5173
3. 重启穿透服务

### 验证页面卡住
**解决方案**：
1. 使用无痕浏览器
2. 清除Cookies
3. 更换浏览器尝试

## ⚡ 快速访问命令

复制以下命令快速启动所有服务：

```powershell

cd backend
mvn spring-boot:run
```

## 🌟 重要提示

1. **保持终端运行**：关闭穿透终端会导致服务中断
2. **耐心等待**：验证后页面加载需要几秒钟
3. **定期维护**：localtunnel免费版可能需要定期重启
4. **外网访问**：必须使用公网域名访问
